# SpringStudentCourse

## Compilation ##
To compile run:
- execute the command: 

``` mvn clean install ```

### Execution ###
To execute look for the artifact in the target folder
- execute the command: 

``` java -jar target/springStudentCourse-RC-1.0.0.jar ```

### Testing ###
To test functionality, you can browse into http://localhost:8080/swagger-ui.html
